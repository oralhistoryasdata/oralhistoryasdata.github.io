Transform Your Interview Transcripts into Interactive Digital Collections

Oral History as Data (OHD) is a free, open-source framework for publishing, analyzing, and visualizing coded oral history interviews.

Powerful Features for Oral History Research

Thematic Coding System

Apply your own analytical codes to transcripts. OHD automatically transforms these into an interactive color-coded visualization that reveals patterns across your interviews.

  • Custom tag creation
  • Visual theme analysis
  • Interactive filtering by theme

CSV-Driven Workflow

Manage all your interview transcripts through simple spreadsheets. Add speaker identifications, thematic tags, and timestamps using familiar tools like Google Sheets or Excel.

  • Simple CSV format
  • Work with familiar spreadsheet tools
  • Batch upload multiple interviews

Media Integration

Connect transcript segments with synchronized audio or video recordings. Support for YouTube, Vimeo, SoundCloud, and MP3 files with timestamp navigation.

  • Timestamp synchronization
  • Multiple media sources supported
  • Jump to specific interview moments

Responsive Design

Your oral history collection works beautifully across all devices - from desktop computers to tablets and smartphones. Provide access to your research wherever your audience is.

  • Mobile-friendly interface
  • Accessible design
  • Built on Bootstrap framework

Simple Workflow from Transcript to Publication

1

Prepare Your Transcripts

Format your interview transcripts into a simple CSV structure with columns for speaker, words, and optional tags. Use familiar tools like Google Sheets or Excel.

2

Set Up Your Repository

Click "Use This Template" on GitHub to create your own project repository. No coding skills required - everything is pre-configured and ready to use.

3

Upload Your Data

Add your transcript CSVs to the repository and create a simple metadata file for each interview. Link to audio/video sources if available.

4

Customize & Publish

Adjust the site configuration, add your project information, and enable GitHub Pages. Your interactive oral history collection is now live on the web!
